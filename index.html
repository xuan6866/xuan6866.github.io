山东大学计算机科学与技术学院 云计算技术 课程实验报告

学号：202200130129 姓名：王旻轩 班级： 智能班
实验题目：利用云平台搭建个人博客
实验学时：2 实验日期：2025.3.12

实验目的：本实验旨在通过实践操作，掌握基于云计算环境的网站部署技能。

具体包括：构建环境并实现个人博客/网站的搭建，撰写实验报告，并将这份报告作为一篇博文发布在自己的博客 / 网站上。

硬件环境： 

联网的计算机一台

软件环境：

Windows

实验步骤与内容：（部分步骤在过程中忘记截图了，故没有进行展示）

1.从网上查阅资料，确定通过Hexo+GitHub的方式来搭建一个自己的个人博客。

2.下载并安装Node.js和Git。

3. 创建并添加SSH 密匙以及验证连接。

4. 创建 Github Pages 仓库，创建后默认自动启HTTPS，博客地址为：https://xuan6866.github.io

5. 本地安装 Hexo 博客程序：

先新建一个文件夹用来存放Hexo的程序文件，然后打开该文件夹，右键，选择Git Bash Here。之后使用 npm 一键安装 Hexo 博客程序，等安装好后再进行初始化并安装所需组件以及启动本地服务器进行预览。

6. 访问 http://localhost:4000，出现如下Hexo默认页面，安装成功。

7. 部署 Hexo 到 GitHub Pages，完成个人博客的搭建。

8. 撰写本次的实验报告，并将这份报告作为一篇博文发布在刚才建立好的博客上：

结论分析与体会：

在本次实验过程中，我通过自己的学习，采用Hexo+GitHub的方式，构建了相关环境并实现了个人博客的搭建。此外我还将这次的实验报告作为第一篇博客发表了上去。通过这次实验学到了许多知识，总体而言收获颇多。
